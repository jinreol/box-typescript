
Section1. 시작하기
==================
<br>

> 타입스크립트 설치  
```console
$ npm install -g typescript 
```

<br>

> 타입스크립트 버전 확인
```console
$ tsc --v 
```

<br>

> 예제 코드 작성 
```ts
// 01-helloworld.ts
let message: string = 'Hello, World'
console.log(message)
```

<br>

> 결과
```console
$ box-typescript node dist/01-helloworld.js 
Hello, World
```
